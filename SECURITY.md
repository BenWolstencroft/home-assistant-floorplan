# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the Floorplan integration or card, please email ben.wolstencroft@example.com with:

- Description of the vulnerability
- Steps to reproduce it
- Potential impact
- Suggested fix (if any)

Please do not open public issues for security vulnerabilities. We will work to address the issue as quickly as possible and provide a public disclosure once a fix is available.

## Supported Versions

We support security updates for the latest version of the Floorplan integration and card.
